Task 2 Maps

Maps

Use a map to store the names of 5 students with their corresponding average grade. You can initialise this map with fictitious hardcoded names and grades.

Write the JavaScript to display the name of each student in a drop-down menu. When the students name is selected from the drop-down menu, the students grade should be displayed using an alert. (Hint: Remember that you can use the value and innerHTML attributes of an option element.)
